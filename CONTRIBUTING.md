# 🤝 Contributing to Macmask

First of all, thanks for taking an interest in this project! Whether you are a veteran hacker or a beginner, your help is welcome.

## 🚀 How can you help?

1.  **Report Bugs**: If the script crashes or doesn't recognize an interface, open an *Issue*.
2.  **Add Features**: Want to add support for other OS or automate more network tasks? Submit a *Pull Request*.
3.  **Code Cleanup**: If you see messy code, feel free to suggest a cleaner way to write it.

## 📜 Guidelines

*   **Keep it Simple**: Macmask is meant to be lightweight. No bloated libraries.
*   **Respect the Style**: Keep the "Ghost" aesthetic and the English logs.
*   **Be Ethical**: No malicious features will be accepted.

## 🛠️ Getting Started

1.  Fork the repository.
2.  Create a new branch (`git checkout -b feature-cool-idea`).
3.  Commit your changes (`git commit -m 'Added something awesome'`).
4.  Push to the branch (`git push origin feature-cool-idea`).
5.  Open a Pull Request.
